## Screenshots Folder

Save screenshots of dashboards you like in `screenshots/` for quick reference during design.

**How to add screenshots:**
1. Open the links from the table above
2. Take a screenshot (Cmd+Shift+4 on macOS, Win+Shift+S on Windows)
3. Save to `screenshots/` with name `{source}_{description}.png`

Examples: `dribbble-ev-management.png`, `flipturn-depot.png`, `behance-elecharge.png`
