## Папка screenshots

Скриншоты понравившихся дашбордов сохраняйте в `screenshots/` для быстрого доступа при проектировании.

**Как добавить скриншоты:**
1. Откройте ссылки из таблицы выше
2. Сделайте скриншот (Cmd+Shift+4 на macOS, Win+Shift+S на Windows)
3. Сохраните в `screenshots/` с именем `{источник}_{описание}.png`

Примеры: `dribbble-ev-management.png`, `flipturn-depot.png`, `behance-elecharge.png`
